# Circle Mini Marketplace - Frontend

Modern React frontend for the Circle Mini Marketplace platform.

## Tech Stack

- React 19
- Vite
- Tailwind CSS
- Framer Motion

## Development

```bash
npm install
npm run dev
```

Runs on `http://localhost:5173`

## Build

```bash
npm run build
```

## Structure

```
src/
├── components/     # React components
├── services/       # API integration
├── assets/         # Images and static files
└── App.jsx         # Main application
```

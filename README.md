"# Circle Mini Marketplace

A professional reselling marketplace platform built with React, Express.js, and Supabase.

## Overview

Circle Mini Marketplace is a full-stack web application that enables buyers and sellers to connect through a clean, intuitive interface. Sellers can list products with images, descriptions, and pricing, while buyers can browse, search, and contact sellers directly.

## Tech Stack

- **Frontend**: React 19, Vite, Tailwind CSS, Framer Motion
- **Backend**: Express.js, Node.js
- **Database**: Supabase (PostgreSQL)
- **Authentication**: Supabase Auth with JWT
- **Storage**: Supabase Storage for product images

## Features

- Role-based authentication (Buyer/Seller)
- Product listing with image upload
- Search and category filtering
- Responsive desktop-optimized design
- Secure API endpoints with JWT validation
- Row Level Security (RLS) for data protection

## Quick Start

1. Clone the repository
2. Set up Supabase project with the provided schema
3. Configure environment variables
4. Start backend: `cd backend && npm run dev`
5. Start frontend: `cd mini-marketplace && npm run dev`

For detailed setup instructions, see [SETUP_GUIDE.md](./SETUP_GUIDE.md).
